# 14 Digit DP

This pattern folder contains core DP problems and example files for each problem.

## Problems

- o1: LC 233 - Number of Digit One
- o2: LC 357 - Count Numbers with Unique Digits
- o3: LC 902 - Numbers At Most N Given Digit Set
- o4: LC 1012 - Numbers With Repeated Digits
- o5: LC 2376 - Count Special Integers
- o6: LC 600 - Non-negative Integers without Consecutive Ones
- o7: LC 788 - Rotated Digits
- o8: CF 276C - Little Girl and Maximum Sum
- o9: CF 1036C - Covered Points Count
- o10: LC 2999 - Count the Number of Powerful Integers
